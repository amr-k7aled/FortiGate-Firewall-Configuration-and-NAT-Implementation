**Project Summary**

This project demonstrates the configuration of a **FortiGate Firewall** in a simulated network environment to provide secure communication between internal and external networks. It includes the complete setup of network interfaces, firewall policies, routing, and **Network Address Translation (NAT)** to enable internet access while protecting the internal network.

The project focuses on applying fundamental **network security** concepts and FortiGate administration practices. It verifies that traffic is securely controlled through firewall rules, translated using NAT, and routed correctly between networks. This hands-on lab provides practical experience in configuring enterprise firewall solutions and troubleshooting connectivity issues.

**Skills Demonstrated:**

* FortiGate Firewall Configuration
* Firewall Policy Management
* Network Address Translation (NAT)
* Routing Configuration
* LAN/WAN Interface Setup
* Network Security Fundamentals
* Connectivity Testing & Troubleshooting
* FortiOS Administration
